class IEEE80211.Dataframe(*arg, **kwargs)¶

Bases: pypacker.pypacker.Packet

DataFrames need special care: there are too many types of field combinations to create classes for everyone. Solution: initiate giving lower type via constructor. In order to use “src/dst/bssid” instead of addrX set from_to_ds to one of the following values:

[Bit 0: from DS][Bit 1: to DS] = [order of fields]

00 = 0 = dst, src, bssid 01 = 1 = bssid, src, dst 10 = 2 = dst, bssid, src 11 = 3 = RA, TA, DA, SA
